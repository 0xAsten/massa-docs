# Node and client configuration

:::caution
Use with caution, overriding some configurations could lead to node
instability and/or a complete desynchronization from Massa blockchain.
:::

## Node configuration

You can override the default configuration via
the `massa-node/config/config.toml` file.

{/* The content will be added at build time via ./scripts/download-configs.sh */}
```toml
EXTERNAL_NODE_CONFIG_CONTENT

```

## Node bootstrap whitelist configuration

You can override the default configuration via
[massa-node/config/bootstrap_whitelist.json]{.title-ref} file.

{/* The content will be added at build time via ./scripts/download-configs.sh */}
```toml
EXTERNAL_BOOTSTRAP_WHITELIST_CONTENT

```

## Client configuration

You can override the default configuration via
the `massa-client/config/config.toml` file.

{/* The content will be added at build time via ./scripts/download-configs.sh */}
```toml
EXTERNAL_CLIENT_CONFIG_CONTENT

```
