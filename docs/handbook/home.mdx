## Identifying Your User Type

Understanding your user type will help you navigate this guide more efficiently. Here's a quick rundown:

- **Non-Technical Users**: Prefer a visual interface and may not have programming experience.
- **SDK Developers**: Have programming experience and prefer to use Massa's SDKs for more customized interactions.
- **API Developers**: Advanced users who interact directly with Massa nodes or Massa Station APIs.

Choose the section that best describes you to get started!
